# diff-review.nvim

A Neovim plugin for reviewing git diffs and pull requests.

## Installation

```lua
-- Using lazy.nvim
{
  'diff-review.nvim',
  config = function()
    require('diff-review').setup({
      -- Configuration options
    })
  end
}
```
